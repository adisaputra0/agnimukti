# Admin Dashboard UI

Branch ini berisi pengembangan dan implementasi antarmuka (UI) untuk halaman Admin Dashboard pada sistem Agnimukti Kremasi Bali.

## Deskripsi

Admin Dashboard digunakan sebagai pusat pengelolaan data dan monitoring sistem oleh administrator. Fokus pengembangan pada branch ini adalah membangun tampilan yang modern, responsif, dan konsisten dengan identitas visual Agnimukti menggunakan color palette **Sacred Earth**.

## Fitur yang Dikerjakan

### Dashboard
- Statistik ringkasan data
  - Total Admin
  - Total Pemohon
  - Total Paket Layanan
  - Total Pendaftaran

- Tabel Pendaftaran Terbaru
  - Kode Pendaftaran
  - Nama Almarhum
  - Status Pendaftaran
  - Tanggal Pendaftaran

- Tabel Status Pembayaran
  - Kode Transaksi
  - Total Pembayaran
  - Metode Pembayaran
  - Status Pembayaran

### Paket Layanan
- Daftar paket kremasi
- Kategori paket:
  - Standar
  - Premium
  - Tambahan
- Informasi harga layanan
- Card layout responsif

### Sidebar Navigasi
- Dashboard
- Pengguna
- Paket Layanan
- Pendaftaran
- Pembayaran
- Pengaturan

### Topbar
- Judul halaman
- Informasi administrator
- Notifikasi

## Teknologi

- PHP
- HTML5
- Tailwind CSS v4
- Tabler Icons
